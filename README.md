# 專案名稱:買賣交易Your Turn to Use

## 專案介紹: 一般賣場以及家裡用不到的東西進行買賣交易

## 功能列表:
### 使用者帳號系統:
#### 登入頁面
##### 會員登入.會員註冊
- (a)登入: 會員帳號.會員名字 (帳號:default  密碼:123).時間顯示
- (b)註冊: 需輸入會員帳號.會員名字.會員密碼.會員密碼確認.電話.地址	
### 使用者操作功能:
- 賣場選擇進入平台商品或二手商品
#### 平台商品
##### 平台現有商品結算金額
- 提供商品單價&確認訂購&時間
- 顯示所有訂單內容
#### 二手商品
##### 二手平台頁面
顯示產品資訊
結算總額
購物結帳
按鈕: 確認結帳.回上一頁.離開 .回我的賣場
#####  賣家頁面 
###### 我的賣場訊息
- (a)賣場訊息: 提供所有商品明細
- (b)新增商品: 輸入產品名.價格.數量.內容; 提供列印功能  
- (c)編輯修改商品資訊: 查詢現況.修改更新商品內容.刪除商品項目
